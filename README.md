# OpenXC Admin UI

## What is this?

A web interface that connects with the database of XtreamCodes 2.9 for admins

#### Features:

- Add Load-Balancer
- Add Streams
- Add Bouquets

## How to install

```
apt install python -y
wget https://raw.githubusercontent.com/tweakunwanted/xc-2.9-admin/master/install.py 
python install.py
```
